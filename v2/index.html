ProteinIdentificationc back to ToC or Class ToC

IRI: http://schema.oceanproteinportal.org/v2/ProteinIdentification

A protein identification made based on information of a peptide identification. This entity serves as an information object used within a Dataset to identify and describe an actual Protein. Where a Protein has a single amino acid sequence that will not vary across datasets submitted to the OPP, th ereported protien names, molecular weights, etc may slightly vary across these datasets. ProteinIdentification exists so that individual datasets can report their findings and remain separate from other ProteinIdentification while still describing the exact same Protein with the same amino acid sequence.
has super-classes
AminoAcidIdentification c
has sub-classes
BestProteinIdentification c
is in domain of
describesProtein op, proteinName dp, sampleProteinProperty op
